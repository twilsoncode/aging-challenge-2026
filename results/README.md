# Results Directory Overview

This directory contains the analytical outputs, performance reports, and interpretability metrics generated by the final chronological age prediction pipeline (`paper_script_v6.py`). 

---

## 📊 Final Model Evaluation Report (`final_model_evaluation_report.csv`)

This file contains the comprehensive evaluation metrics for all trained models. It tracks Mean Absolute Error (MAE), Root Mean Squared Error (RMSE), R-squared (R²), Pearson correlation, and Spearman correlation. 

The report assesses the 8 individual base learners on the OneK1K test set (Step 1), as well as the cross-dataset generalization of the 3 final Ridge Meta-Ensembles on both the OneK1K and AIDA test sets (Step 2).

## 🧬 Full Feature Rankings (`full_feature_rankings_step1.csv`)

This file is a compiled master list containing the top-to-bottom ranked importance of every feature used by the 8 individual base models during **Step 1** of the pipeline.

Because the models rely on fundamentally different mathematical architectures and input modalities, this file consolidates two different ranking methodologies:
1. **Native Feature Weights:** Extracted directly from the algorithms of the tree-based and linear models (XGBoost, LightGBM, Random Forest, ElasticNet) scoring the importance of the OneK1K Pseudobulk genes.
2. **Permutation Importance:** Calculated manually for the neural network and distance-based "black box" models (MLP, KNN) scoring the impact of the Geneformer and scGPT embedding dimensions on the test set.

### Structure of the CSV
* `Feature`: The name of the specific gene or embedding dimension (e.g., `ENSG00000120162__CD4_T`, `Geneformer_dim_120`, `sex_binary`).
* `Importance`: The numerical magnitude score assigned to that feature.
* `Model`: Which of the 8 base models the feature was ranked for.
* `Rank`: The feature's positional rank (1 being the most important) for that specific model.
